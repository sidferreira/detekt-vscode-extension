# Changelog

All notable changes to the Detekt VSCode Extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.9] - 2024

### Added
- Initial release with core functionality

### Features
- Auto-run detekt on Kotlin file save
- Manual command to run detekt on entire project
- Problems panel integration
- Configurable detekt executable path
- Support for additional detekt arguments

### Configuration
- `detekt.enable`: Enable/disable the extension
- `detekt.runOnSave`: Auto-run on file save
- `detekt.executablePath`: Custom detekt binary path
- `detekt.args`: Additional CLI arguments

## [Unreleased]
